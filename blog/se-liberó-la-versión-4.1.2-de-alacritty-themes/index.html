Se liberó la versión 4.1.2 de Alacritty Themes

by Juan Vásquez, 21 Sep 2021
alacritty, alacritty-themes, javascript

Recuerdan que unos días atrás publiqué un artículo donde comencé a contribuir al proyecto de Alacritty-Themes, bueno a partir de ahí muchas cosas han pasado.

Alacritty-Themes versión 4.1.2

Rajasejar Chandran el creador de este paquete me invitó a ser parte del proyecto y apartir de ahí unimos esfuerzos para resolver algunos problemas que se habían tenido en la librería, sin duda, el más importante que se resolvió era el hecho que cuando se actualizaba el archivo alacritty.yml del usuario se borraban TODOS los comentarios que el usuario tenía en su archivo YAML y esta liberación arregla eso.

Te invito a probar Alacritty-Themes 🌈😍

espero seguir contribuyendo en este proyecto pues me agrada mucho! 🚀
